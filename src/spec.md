# Specification

## Summary
**Goal:** Build FF PAYALUGA, a Free Fire Clash Squad 1v1 tournament platform with user registration, wallet-based deposits, tournament joining, and admin management.

**Planned changes:**
- Implement user authentication (email/password) with auto-generated unique User IDs
- Create wallet system with coin balance tracking and deposit functionality (₹10-₹100 range)
- Display Google Pay QR code for deposit requests
- Build admin panel for manual deposit verification and coin addition (1₹ = 1 Coin)
- Implement tournament join system requiring Free Fire UID and WhatsApp number, with automatic entry fee deduction
- Send join details instantly to admin dashboard
- Create admin controls for user management, tournament configuration (entry fee, match time, schedule), and status management (Open/Closed)
- Design Free Fire gaming themed dark UI with animated buttons and esports layout
- Ensure full mobile responsive design
- Create six-page structure: Home, Login/Register, Dashboard, Wallet/Deposit, Tournament Page, Admin Panel

**User-visible outcome:** Users can register, deposit coins via Google Pay, join Free Fire Clash Squad 1v1 tournaments using wallet coins, and admins can manage deposits, tournaments, and view all player details through a comprehensive admin panel—all within a dark-themed, mobile-friendly gaming interface.
